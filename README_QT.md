- # LCFT Qt 6.10 Client
  
  This project contains a Qt 6.10 QML client for the LCFT LAN/WAN file transfer system.
  
  ## Features
  
  - Windows and Android client support
  - LAN device discovery
  - LAN file transfer
  - WAN relay via ECS server
  - Qt/QML graphical interface
  - Mobile UI for Android and narrow windows
  
  ## Build
  
  1. Open `CMakeLists.txt` in Qt Creator.
  2. Select a Qt 6.10 Desktop kit or Android kit.
  3. Build and run `appLanChatShell`.
